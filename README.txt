Programa feito por George Carlos-Aluno de LP1 da turma de Samuel

Notas Necessárias:
1-Formato do arquivo "estoque.csv" foi ajustado para dividir os atributos com espaçamento normal
e sem aspas ou vírgula, foi também modificado as vírgulas dos valores para ponto, para maior
facilidade na captura dos valores de cada produto. 
OBS:Cuidado ao passar o codigo do produto, pois se passar um string confundindo com o nome do produto, o programa irá entrar em loop
e terá que matar o programa pra parar.

2-Foi feito um Makefile para que possa manipular os arquivos com mais prática;
Logo basta executar 'make' no terminal na pasta em que está os arquivos e ele irá
criar os arquivos.o para compilar.

3-Para executar o programa, deve-se executar o comando './supermercado' no terminal;
Logo irá surgir um menu interativo no terminal para manipular o programa.
OBS: 'make clean' pode ser executado para limpar os arquivos.o e o executável do programa.

4-Programa feito pela IDE CLion, na plataforma WSL do windows rodando um subsistema Linux Ubuntu;
Logo pode apresentar falhas de formatação(o que já aconteceu em outros projetos dependendo do sistema em que rodou);
